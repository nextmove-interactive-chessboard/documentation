\chapter*{Kurzfassung /Abstract }
\label{cha:abstract}

Diese Diplomarbeit beschäftigt sich mit der Entwicklung eines digitalen, einsteigerfreundlichen Schachbretts, das ohne komplexe Vorkenntnisse den Einstieg in das Schachspiel erleichtert. Das System erkennt die vom Spieler ausgeführten Züge, zeigt zulässige Zugmöglichkeiten visuell auf dem Spielbrett an und unterstützt den Spieler durch eine integrierte Spielanalyse, welche auf fehlerhafte oder ungünstige Züge hinweist. Ziel des Projekts ist es, eine intuitive Lernhilfe zu schaffen, die insbesondere Anfängern hilft, die Spielregeln zu verstehen und ihre Fähigkeiten schrittweise zu verbessern.

Das Konzept basiert auf einer eigens entwickelten Hardware in Kombination mit einer darauf abgestimmten Softwarelösung. Ein zentraler Bestandteil der Arbeit ist die Konstruktion und das Design des Schachbretts sowie die Programmierung der Steuer- und Auswerteeinheiten. Die lokale Verarbeitung der Spieldaten ermöglicht eine direkte Rückmeldung während des Spiels. Ergänzend dazu wurde eine Serveranbindung realisiert, über die Spielstände und Analysedaten gespeichert werden. Erste Tests zeigten, dass durch die Kombination aus visueller Zugunterstützung und automatischer Analyse ein effektiver Lernprozess gefördert werden kann.

Trotz auftretender Herausforderungen bei der Hardwareintegration und der Kommunikation zwischen lokaler Einheit und Server konnte ein funktionales Gesamtkonzept umgesetzt werden. Zukünftige Weiterentwicklungen könnten eine verbesserte Fehlererkennung, eine erweiterte Spielanalyse sowie eine optimierte Benutzeroberfläche umfassen. Insgesamt bildet diese Arbeit die Grundlage für ein digitales Schachsystem, das moderne Technik nutzt, um den Zugang zum Schachspiel nachhaltig zu erleichtern und neue Möglichkeiten für das spielerische Lernen zu eröffnen.   \\ \\

This diploma thesis focuses on the development of a digital, beginner-friendly chessboard designed to make it easier to get started with the game of chess without requiring extensive prior knowledge. The system detects the moves made by the player, visually displays all legal moves directly on the chessboard, and supports the player through an integrated game analysis that highlights incorrect or unfavorable moves. The aim of the project is to create an intuitive learning aid that helps beginners understand the rules of the game and gradually improve their skills.

The concept is based on custom-developed hardware combined with a tailored software solution. A central part of the thesis is the design and construction of the chessboard as well as the programming of the control and evaluation units. Local processing of game data enables immediate feedback during gameplay. In addition, a server connection was implemented to store game states and analysis data. Initial tests showed that the combination of visual move assistance and automated analysis effectively promotes the learning process.

Despite challenges encountered during hardware integration and communication between the local unit and the server, a functional overall concept was successfully realized. Future developments could focus on improved move detection, extended game analysis, and an optimized user interface. Overall, this thesis provides the foundation for a digital chess system that leverages modern technology to sustainably simplify access to the game of chess and open up new possibilities for interactive learning.\\
