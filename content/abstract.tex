\chapter*{Kurzfassung /Abstract}
\label{cha:abstract}

Diese Diplomarbeit beschäftigt sich mit der Entwicklung eines interaktiven, einsteigerfreundlichen Schachbretts, 
das ohne jegliche Vorkenntnisse den Einstieg in das Schachspiel ermöglicht. Das System erkennt die vom Spieler 
ausgeführten Züge, zeigt zulässige Zugmöglichkeiten visuell per Lichtsignal auf dem Spielbrett an und unterstützt 
den Spieler durch eine Spielanalyse auf einer Website. Ziel des Projekts ist es, eine intuitive Lernhilfe zu schaffen, 
die insbesondere Anfängern dabei hilft, die Spielregeln zu verstehen und ihre Fähigkeiten schrittweise zu verbessern.

Das Konzept basiert auf einer eigens entwickelten Hardware in Kombination mit einer darauf abgestimmten Softwarelösung. 
Ein zentraler Bestandteil der Arbeit ist die Konstruktion und das Design des Schachbretts sowie die Programmierung der 
Steuer- und Auswerteeinheiten. Die lokale Verarbeitung der Spieldaten ermöglicht eine direkte Rückmeldung während des 
Spiels. Ergänzend dazu wurde eine Serveranbindung realisiert, über die Spielstände und Analysedaten gespeichert werden.

Trotz auftretender Herausforderungen bei der Hardwareintegration und der Kommunikation zwischen lokaler Einheit und 
Server konnte ein funktionales Gesamtkonzept umgesetzt werden. Zukünftige Weiterentwicklungen könnten eine verbesserte 
Fehlererkennung, eine erweiterte Spielanalyse sowie eine optimierte Benutzeroberfläche umfassen. Insgesamt bildet 
diese Arbeit die Grundlage für ein digitales Schachsystem, das moderne Technik nutzt, um den Zugang zum Schachspiel 
nachhaltig zu erleichtern und neue Möglichkeiten für das spielerische Lernen zu eröffnen.   \\ \\

\newpage

This thesis deals with the development of an interactive, beginner-friendly chessboard that allows players to start 
playing chess without any prior knowledge. The system recognizes the moves made by the player, visually displays 
possible moves on the game board using light signals, and supports the player by analyzing the game on a website. 
The aim of the project is to create an intuitive learning aid that helps beginners in particular to understand the 
rules of the game and gradually improve their skills.

The concept is based on specially developed hardware in combination with a customized software solution. A central 
component of the work is the construction and design of the chessboard as well as the programming of the control and 
evaluation units. Local processing of the game data enables direct feedback during the game. In addition, a server 
connection has been implemented to store game scores and analysis data.

Despite challenges encountered in hardware integration and communication between the local unit and the server, a 
functional overall concept was implemented. Future developments could include improved error detection, enhanced 
game analysis, and an optimized user interface. Overall, this work forms the basis for a digital chess system that 
uses modern technology to make chess more accessible in the long term and open up new opportunities for playful 
learning. \\
